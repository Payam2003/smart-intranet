# SMART-INTRANET
Progetto di tirocinio svolto presso l'azienda PERFORMER SRL. Sviluppato come PWA (Progressive Web App) con integrazione dei servizi Firebase.
SOLO LE PERSONE AUTORIZZATE DELLA INTRANET POSSONO ACCEDERVI.

Per avviare il progetto in locale (in fase di sviluppo), eseguire dal path `/pwa`, in un terminale il comando:
`npx vite --port=4001`
E da un altro terminale dal path `smart-intranet`, il comando:
`firebase emulators:start`

-----------------

Per avviare il progetto in produzione, eseguire da terminale il comando:
`firebase deploy`

-----------------
